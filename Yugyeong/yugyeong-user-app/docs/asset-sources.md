## Seoul Hangang Font

- Font Name: Seoul Hangang
- Files:
  - SeoulHangangL.ttf
  - SeoulHangangM.ttf
  - SeoulHangangB.ttf
  - SeoulHangangEB.ttf
- Source: Seoul Metropolitan Government Seoul Typeface
- Download Page: https://english.seoul.go.kr/seoul-views/seoul-symbols/5-fonts/
- Note: 서울서체는 무료 사용 가능하지만, 폰트 파일의 판매·유료 양도 및 허가 없는 재배포는 금지되어 있으므로 저장소에는 포함하지 않는다.

## Restore Fonts

1. Download Seoul Typeface from the official Seoul Metropolitan Government page.
2. Extract Seoul Hangang font files.
3. Place these files under:

public/fonts/

Required files:
- SeoulHangangL.ttf
- SeoulHangangM.ttf
- SeoulHangangB.ttf
- SeoulHangangEB.ttf
